# Pseudopotentials

Fully relativistic ONCVPSP (Optimized Norm-Conserving Vanderbilt Pseudopotentials)
from the [PseudoDojo](http://www.pseudo-dojo.org/) project, generated by D.R. Hamann
using ONCVPSP v3.3.0.

- `standard/` — Standard accuracy (suitable for most GW calculations)
- `stringent/` — Higher accuracy (harder cutoffs, more projectors)

Both sets are PBE exchange-correlation, fully relativistic (`relativistic="full"`),
compatible with QE `noncolin=.true., lspinorb=.true.` calculations.

Source: `/home/jackm/SOURCES/agent_tasks/qe-gw-pipeline/assets/`

## Usage

In QE input files, set `pseudo_dir` to either directory, or symlink the needed
element files into the run directory. The `skills/build_inputs/SKILL.md` covers this.
